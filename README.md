# BillSplitting

Split bills into itemized receipts.

This is an Epicodus group project for Java that focuses on API development.

## Getting Started



### Prerequisites

You will need the following packages and software installed:

```
Java JDK 1.8.0_151
JAVA JRE 1.8.0_151
IntelliJ IDEA
```
